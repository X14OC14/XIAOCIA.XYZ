▸ HOW TO USE
1
Atur layer & biome
Gunakan panel LAYERS dan BIOME di sebelah kanan. Bisa pakai preset atau custom sendiri.
2
Kasih nama worldnya
Isi nama di field World Name lalu klik DOWNLOAD .mcworld. File langsung generate di browser, gak perlu internet.
3
Import ke Minecraft
Buka file .mcworld dari file manager → Minecraft otomatis import → world siap dimainkan! 🎮
4
Tidak bisa import?
Coba rename file jadi .zip → extract → taruh di folder com.mojang/minecraftWorlds/ secara manual.
